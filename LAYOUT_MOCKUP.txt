NOTES ON LAYOUT:
================

1. The top section (Name through Notes) mirrors the Roll20 vehicle sheet
   field-for-field. If we copy the Roll20 sheet, we can reuse its CSS
   with the floor plan appended below.

2. The floor plan section is split:
   - LEFT: System Key/Legend showing each placed system type with its
     color swatch, space count, and CP value
   - RIGHT: The canvas grid where systems are placed visually

3. Each canvas cell = 1 system space. The grid auto-sizes based on
   chassis selection. Unused spaces show as a subtle pattern (░).

4. Systems placed on the canvas automatically populate the systems
   table above. Editing a system in the table updates the canvas
   and vice versa.

5. The floor plan canvas supports:
   - Click palette item → click canvas to place
   - Drag to move placed systems
   - R to rotate (for non-square systems)
   - Right-drag to pan
   - Scroll to zoom
   - Click system in legend to highlight on canvas
   - Wall/door drawing on cell edges (future)

6. Export options at the bottom include saving the floor plan as a
   separate PNG image.

7. The Modifiers Reference table stays at the very bottom as a
   quick lookup.

8. For the standalone web version, all sections are on one scrollable
   page. For Roll20 integration, the floor plan section would be
   appended to the existing vehicle tab.
