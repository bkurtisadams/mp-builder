Using the Voyage Map

Setup: Choose origin and destination ports, add legs to build a route. Each leg can have a different water type.

Hexes: Each hex is 6 miles across. Click any hex to see its coordinates. The ship icon shows current position.

Advancing Days: Switch to the Log tab and click Advance Day. Each press simulates one day of sailing — weather is rolled, encounters checked, distance calculated.

Events: Coloured markers appear on the map where encounters or damage occurred. Click a marker for details.

Map Image: Place a file named greyhawk-map.jpg in the same folder to use a background map image. The hex grid is calibrated to standard Greyhawk geography.
